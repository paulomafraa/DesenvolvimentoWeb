O que é JavaScript?
O JavaScript é uma linguagem de programação de alto nível, interpretada e multiparadigma, que se tornou a tecnologia padrão para criar interatividade e comportamentos dinâmicos em páginas web do lado do cliente.

É a mesma coisa que Java? Se não, quais as semelhanças e quais as diferenças?
Não, são linguagens distintas; enquanto o Java é uma linguagem compilada, fortemente tipada e que roda em uma máquina virtual (JVM), o JavaScript é uma linguagem de script interpretada pelos navegadores. Elas compartilham apenas uma semelhança sintática básica baseada na linguagem C, mas possuem propósitos e arquiteturas completamente diferentes.

Quais são as utilizações mais comuns de JavaScript em HTML básico?
No contexto de HTML básico, o JavaScript é frequentemente utilizado para a validação de dados em formulários antes do envio, criação de alertas e janelas modais, manipulação de menus responsivos e alteração de conteúdos ou estilos CSS sem a necessidade de recarregar a página.

Qual o nome da notação de objetos em JavaScript? Como ela funciona?
O nome da notação é JSON (JavaScript Object Notation). Ela funciona através de uma estrutura de dados baseada em pares de "chave: valor" organizados dentro de chaves { }, sendo amplamente utilizada para o intercâmbio de dados entre sistemas.

Como JavaScript faz referência a objetos? Qual a diferença entre elemento.nome e elemento["nome"]?
O JavaScript faz referência a propriedades de objetos de duas formas: a notação de ponto (elemento.nome) é a mais utilizada pela sua legibilidade e rapidez, enquanto a notação de colchetes (elemento["nome"]) é indispensável quando o nome da propriedade é dinâmico (armazenado em uma variável) ou contém caracteres que não seriam permitidos na notação de ponto, como espaços ou hífens.

O documento HTML pode ser considerado como um conjunto de objetos? É possível manipulá-lo via JavaScript? Como?
Sim, o documento HTML é interpretado pelo navegador como uma árvore de objetos conhecida como DOM (Document Object Model). É perfeitamente possível manipulá-lo via JavaScript através de métodos de seleção (como document.getElementById ou querySelector) que permitem alterar textos, cores, atributos e a própria estrutura das tags em tempo real.
