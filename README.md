# DesenvolvimentoWeb

Repositório para os trabalhos da disciplina eletiva Desenvolvimento Web.

## Pastas

| Pasta | Trabalho |
|-------|----------|
| `tabelas/` | Trabalho com tabelas HTML |
| `formulario-html/` | Formulário HTML (Feira de Tecnologia 2026) |

### Como abrir

- Tabelas: abra `tabelas/pagina1.html` (ou `pagina2.html` / `pagina3.html` / `comcss.html`) no navegador
- Formulário: abra `formulario-html/index.html` no navegador
